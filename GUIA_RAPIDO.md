# 🚀 Guia Rápido de Início - Igreja São Sebastião

## ⚡ Início Rápido (3 minutos)

### 1. Instalar Dependências
```bash
pip install -r requirements.txt
```

### 2. Configurar Senha (Opcional mas Recomendado)
Edite o arquivo `.env` e mude:
```
ADMIN_PASSWORD=sua_senha_aqui
```

### 3. Iniciar o Sistema
```bash
python app.py
```

### 4. Acessar
- **Site:** http://localhost:5000
- **Admin:** http://localhost:5000/admin
- **Login:** admin / admin123 (ou sua senha do .env)

---

## 📝 Tarefas Iniciais

### Ao fazer o primeiro acesso:

#### 1️⃣ Configurações Básicas (5 min)
1. Acesse **Configurações** no menu admin
2. Atualize:
   - ✏️ Nome da Igreja
   - ✏️ Telefone
   - ✏️ Email
   - ✏️ Endereço
   - ✏️ Redes Sociais

#### 2️⃣ Informações da Paróquia (10 min)
1. Acesse **Informações**
2. Edite cada seção:
   - 📖 Sobre a Paróquia
   - 📜 Nossa História
   - 🎯 Nossa Missão
   - ✨ Hero (Frase principal)

#### 3️⃣ Horários de Missas (5 min)
1. Acesse **Horários de Missas**
2. Edite os horários existentes ou
3. Adicione novos horários

#### 4️⃣ Adicionar Notícias (5 min)
1. Acesse **Notícias e Eventos**
2. Clique em **Nova Notícia**
3. Adicione pelo menos 2-3 notícias

#### 5️⃣ Galeria de Fotos (10 min)
1. Acesse **Galeria de Fotos**
2. Adicione fotos da igreja
3. Recomendado: 6-12 fotos iniciais

#### 6️⃣ Fazer Backup (1 min)
1. No **Dashboard**
2. Clique em **Fazer Backup**
3. Salve o arquivo em local seguro

## 🎨 Dicas de Uso

### Para Adicionar Conteúdo de Qualidade:

**Notícias:**
- Use títulos chamativos mas informativos
- Adicione sempre uma imagem (recomendado: 1200x630px)
- Escreva pelo menos 2-3 parágrafos
- Use o tipo "Evento" para eventos futuros

**Fotos:**
- Resolução mínima: 1024x768px
- Tamanho máximo: 5MB
- Prefira fotos horizontais
- Evite fotos muito escuras

**Textos:**
- Seja claro e objetivo
- Use parágrafos curtos
- Revise a ortografia
- Atualize regularmente

## 🆘 Problemas Comuns

### "Não consigo fazer login"
- Verifique usuário: `admin`
- Senha padrão: `admin123`
- Ou a senha que você definiu no `.env`

### "Erro ao subir imagem"
- Tamanho máximo: 5MB
- Formatos: JPG, PNG, GIF, WEBP
- Reduza o tamanho se necessário

### "Site não carrega"
- Verifique se o servidor está rodando
- Acesse: http://localhost:5000
- Veja erros no terminal
